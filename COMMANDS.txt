Tips & Tricks
================================================================

1. เช็ค config ก่อนรันเสมอ:
   E:\SS5_internship\env_job\Scripts\python.exe pipeline_runner.py --dry-run

2. ใช้ verbose เมื่อ debug:
   E:\SS5_internship\env_job\Scripts\python.exe pipeline_runner.py -v

3. Backup config ก่อนแก้ไข:
   copy config.json config.json.backup

4. ดู log file เมื่อเจอปัญหา:
   type logs\pipeline.log

5. ทดสอบกับ input file เล็กๆ ก่อน:
   - แก้ data_paths.input_file ใน config.json
   - ชี้ไปที่ไฟล์ทดสอบขนาดเล็ก
